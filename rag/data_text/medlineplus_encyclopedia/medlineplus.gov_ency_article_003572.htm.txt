Schilling test
To use the sharing features on this page, please enable JavaScript.
The Schilling test can determine whether the body absorbs
vitamin B12
normally. It helps assess the cause of vitamin B12 deficiency. This test is rarely, if ever, done today.
Alternative Names
Vitamin B12 absorption test
